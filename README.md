# Booktracker

# Objectives

    create a user friendly platform.
    Add a list of books.
    Delete a book

# Technology used

    React
    Javascript
    CSS

# Instruction

## github

    https://github.com/antonyosebe/book-tracker.git
    run npm install
    run npm start

## server

    https://github.com/antonyosebe/book-tracker-server.git
    run bundle install
    run bundle exec server
